## API

| 属性  | 说明     | 类型                                                              | 默认值 | 必填    |
| ----- | -------- | ----------------------------------------------------------------- | ------ | ------- |
| value | 图标名称 | `'addcart' | 'alipay' | 'close' | 'remind' | 'selected' | string` | -      | `true`  |
| size  | 尺寸     | `'xs' | 's' | 'm' | 'l' | 'xl'`                                   | `'l'`  | `false` |
